Цели и задачи проекта: 
- закрепить навыки анализа данных
- выполнить обработку данных для анализа
- найти и отбросить выбросы числовых переменных
- оценить влияние номинативных переменных
- отобрать переменные для дальнейшего построения модели

Информация о данных:
в датафрейме присуствуют численные и номинативные переменные
в переменных присутствуют пропуски и выбросы

Этапы работы над проектом:
- изучение датафрейма, в т.ч. уникальных значений для каждой переменной
- предварительная обработка данных
- анализ числовых переменных
- анализ номинативных переменных
- выводы
- выбор переменных для дальнейшего построения модели
